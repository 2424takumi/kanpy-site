# Kanpy site

`https://2424takumi.github.io/kanpy-site/` で公開される Kanpy の Web フロント:

- `/` (`index.html`) — マーケティング LP
- `/privacy.html` — プライバシーポリシー (`privacy.md` を Jekyll が変換)
- `/terms.html` — 利用規約 (`terms.md` を Jekyll が変換)

## デプロイ

このリポジトリは Public + GitHub Pages 有効化されており、 `main` ブランチへの push で自動公開される。

iOS アプリ本体は別リポジトリ ([2424takumi/kanpy](https://github.com/2424takumi/kanpy)、 private) で管理。
